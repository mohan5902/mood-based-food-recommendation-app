# mood-based-food-recommendation-app
A web application that recommends food based on user mood, hunger level, and weather conditions
## Features
- Mood-based food suggestions
- Hunger & weather aware logic
- Dark mode toggle 
- Saves last selected mood using localStorage
- Clean and responsive UI

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Clone the repository
2. Open `index.html` in browser or Live Server

## Future Enhancements
- AI-based recommendations
- API integration
- Food images
- User authentication
